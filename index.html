How does Flexbox work?

Flexbox is a one-dimensional layout method for arranging items in rows or columns. It governs the distribution of space among items in a container when dealing with varying sizes, often used for navigation bars or card layouts.

What is the DOM?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. JavaScript is the primary language used to manipulate the DOM.
